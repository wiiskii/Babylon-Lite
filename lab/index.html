🌳
100% Tree-Shakable
Zero module-level side effects. Import only what you use — unused features are stripped from the bundle.
⚡
WebGPU Exclusive
No WebGL fallback. Built entirely around modern GPU paradigms: render pipelines, compute shaders, bind groups.
🎯
Pixel-Perfect Parity
Rendering output is mathematically identical to Babylon.js. Validated via automated pixel diff testing.
📦
Minimal Footprint
Dramatically smaller bundles than Babylon.js. Data-oriented design, flat arrays for GPU transfers.
🔌
One-Way Data Ownership
Scene owns everything. Meshes, lights, cameras are plain data — zero circular deps, trivial serialization.
